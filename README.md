PDFly is a free, simple, and fast online PDF editor. It lets you open PDFs in your browser, add text, highlights, drawings, signatures, images, and export the edited file instantly.

## Features
- Open and view PDF files
- Add text, highlights, rectangles, and freehand drawings
- Insert images and signatures
- Undo and clear annotations
- Zoom in/out and navigate pages
- Reorder pages before exporting
- Search & replace text
- Dark mode toggle
- Export as a new PDF with all changes applied

## How to Use
1. Open the [PDFly Website](https://YOUR_USERNAME.github.io/pdfly/).
2. Upload a PDF by clicking **Open PDF** or dragging a file.
3. Select a tool from the toolbar and edit your PDF.
4. Click **Download** to save your edited PDF.

## Deployment
This project is hosted on **GitHub Pages**. To deploy your own copy:
1. Fork or clone this repository.
2. Update `index.html` with your changes.
3. Push to the `main` branch.
4. GitHub Pages will automatically update your live site.
